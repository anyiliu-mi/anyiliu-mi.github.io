When we give AI agents tools—browsers, terminals, API keys—we are effectively giving them agency in the real world. This post explores the "Tool Authority" problem: how do we ensure an agent only uses a tool for its intended purpose?

We discuss the concept of *least privilege* applied to AI, and why capability-based security models might be a good fit for agentic architectures.
